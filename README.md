## Installation

First, clone the repo to your local machine.

```zsh
$ git clone https://github.com/GiovanniDw/functional-programming.git
```

Then `$ cd ` into the project folder.

```zsh
$ npm install
```

Now run the project!

```zsh
$ npm run dev
```
